使用代码编辑器HBuilderX制作网页

HBuilderX，简称HX。 H是HTML的首字母，Builder是构造者，X是HBuilder的下一代版本。
HX既可以是编辑器，也可以是IDE（集成开发环境，使程序员能够整合编写计算机程序的不同方面）。

下载地址：https://www.dcloud.io/hbuilderx.html

支持java插件、nodejs插件，并兼容了很多vscode的插件及代码块。 还可以通过外部命令，方便地调用各种命令行功能，并设置快捷键。

可在网页里引入flourish、tableau、echarts在线数据图。

[参见HX教程] (https://www.dcloud.io/hbuilderx.html)

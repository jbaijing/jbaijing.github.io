在线引入Echarts地图遇到的问题

发表于 2020-12-04

在线地图不符合国家标准问题

在线引入Echarts地图，除了需要引入echarts.min.js，还需要引入china.js，但由于Echarts原有China地图不是国家标准地图，官方已经停用，因此引入Echarts地图会遇到问题。

另外，引入echarts.min.js时，在线引入经常由于在线链接的变化而导致链接失效，导致Echarts无法正常引入。建议下载最新版echarts.min.js，将其置于与index同一文件夹，这样只要写一句引入代码就可以了。

<script src="echarts.min.js">

[参见Echarts教程] (https://echarts.apache.org/zh/tutorial.html)

master改为main的影响

发表于 2020-11-30 更新于 2020-12-01

用hexo和github搭建个人博客踩的最大的坑

2020年10月1日起，在github上创建的任何新仓库都将使用main作为默认分支，而不是master。

因此，在hexo的_config里，要将以下代码做修改

Deploy
type: git
repo: https://github.com/yourname/yourname.github.io.git
branch: main

注意，branch以前是master，现在要改为main
